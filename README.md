# HelpDesk Lite

Sistema completo demonstrativo para meu portfólio sobre abertura e gerenciamento de chamados de suporte técnico.

# 🌐 Demonstração ao vivo
Para testar acesse o link abaixo e faça uma reclamação demonstrativa. Logo após, use o Acesso de Admin para visualizar o painel administrativo e como fica visível os chamados.

[https://helpdesk-lite.onrender.com](https://helpdesk-lite.onrender.com)

# 🔑 Acesso Admin
- **Usuário:** `recrutador`
- **Senha:** `helpdesk2026`

# ⚙️ Funcionalidades

- Abertura de chamados com protocolo
- Painel administrativo
- Status: Novo, Em andamento, Resolvido
- Histórico de chamados
- Login global para demonstração

# 🛠️ Tecnologias

Python • Flask • SQLite • HTML • CSS

```bash
pip install -r requirements.txt
